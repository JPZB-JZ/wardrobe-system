# Aura Style 智能衣橱 - 开发记忆文档

## 项目概述
- **名称**: Aura Style 智能衣橱
- **定位**: 个人衣橱管理 + AI 搭配推荐
- **目标**: 做成 Android APK（通过 Capacitor 打包）

## 技术栈
- **前端**: React 18 + TypeScript + Vite 5 + TailwindCSS 3
- **状态**: Zustand（轻量状态管理）
- **存储**: localStorage（前端离线）→ 后续可接 Express + SQLite
- **AI**: DeepSeek API（衣物识别 + 搭配推荐）
- **打包 APK**: Capacitor 5

## 设计风格
- **Dopamine Aura（柔和版）**
- 主色: `#c46d9c`（玫瑰粉）
- 辅色: `#8e6dbf`（薰衣草紫）
- 大量毛玻璃效果 + 圆角 + 渐变
- 整体舒适柔和，不刺眼

## 核心功能
1. 首页智能搭配推荐（评分系统）
2. 衣橱管理（CRUD + 品类/颜色/场合筛选）
3. 拍照上传 + AI 识别（自动填表单）
4. AI 搭配推荐（DeepSeek）
5. 穿着日历 + 统计分析
6. 数据导入/导出/重置

## APK 打包流程（Capacitor）
```bash
# 1. 构建前端
cd 02-前端源码
npm run build

# 2. 初始化 Capacitor（首次）
npx cap init "Aura Style" "com.aurastyle.app" --web-dir ../05-构建产物
npx cap add android

# 3. 同步构建产物
npx cap sync

# 4. 打开 Android Studio 构建
npx cap open android
# 在 Android Studio 中 Build > Build APK
```
